# String Escape Sequences

| Escape Sequence | Unicode Code Point           |
|-----------------|------------------------------|
| `\a`            | 07                           |
| `\b`            | 08                           |
| `\f`            | 0C                           |
| `\n`            | 0A                           |
| `\r`            | 0D                           |
| `\t`            | 09                           |
| `\v`            | 0B                           |
| `\e`            | 1B                           |
| `\\`            | 5C                           |
| `\'`            | 27                           |
| `\"`            | 22                           |
| `\???`          | 8 bit octal codepoint        |
| `\x??`          | 8 bit hexadecimal codepoint  |
| `\u????`        | 16 bit hexadecimal codepoint |
| `\U????????`    | 32 bit hexadecimal codepoint |
